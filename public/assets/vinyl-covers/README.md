# 🎵 Vinyl Cover Artworks - MusicTwins

## Carpeta para las imágenes de los discos de vinilo

Añade aquí las imágenes que quieras usar como fondo de los discos. Cuando añadas las imágenes, se integrarán automáticamente en los discos.

### 📁 Estructura esperada:

```
vinyl-covers/
├── plataforma.jpg       (Para "La Plataforma")
├── caracteristicas.jpg  (Para "Características")
├── conecta.jpg          (Para "Conecta & Comparte")
├── contacto.jpg         (Para "Contacto")
└── README.md            (Este archivo)
```

### 🎨 Especificaciones recomendadas:

- **Formato**: JPG, PNG o WebP
- **Tamaño**: Mínimo 300x300px, ideal 600x600px
- **Proporción**: Cuadrada (1:1)
- **Estilo**: Artístico, musical, profesional
- **Paleta de colores sugerida**:
  - **Plataforma**: Tonos marrón/earth tones
  - **Características**: Tonos gris/platino
  - **Conecta**: Tonos naranja/warm
  - **Contacto**: Tonos azul/cool

### 💡 Cómo integrar:

1. Coloca las imágenes en esta carpeta con los nombres sugeridos
2. El código automáticamente las reconocerá
3. Aparecerán como fondo de cada disco

### 🎼 Alternativa: Usar como descarga

Si prefieres imágenes ya listsas, puedes descargar cover art de:
- Unsplash.com
- Pexels.com
- Pixabay.com
- Spotify (usar bajo licencia)

**Filtros sugeridos**: "Music", "Vinyl", "Album", "Sound", "Beats"

---

**Nota**: Las imágenes se mostrarán en los discos como fondo. Si no existen, se mantienen los diseños con gradientes actuales.
